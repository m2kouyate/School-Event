Import Events

The CSV file should contain the following fields in this order: Name, Date and Time (YYYY-MM-DD HH:MM), Location, Description, Max Participants, Image Filenames (separated by semicolons).

Example:

  Event Name,2023-09-01 14:00:00,New York City Hall,This is a description of the event,100,image1.jpg;image2.jpg

Notes:

  • The images with filenames "image1.jpg" and "image2.jpg" should be included in the file upload section for images.
  • The allowed image formats are: JPG, PNG.
